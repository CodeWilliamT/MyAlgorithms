这个Repo主要包含了个人日常练习算法做的leetcode题目的代码(C++)、做题时的思路注释，一个本人整合编写的设备控制、图像处理的工具类库以及一些个人开发使用的小软件、脚本。

LeetCodeDemo:个人日常Leetcode算法做题Repo。注释包含自身的处理思路。

MyMachineControl_Toolkits:一些硬件控制跟图像处理开发编写使用的类库和一些Demo,C#。
